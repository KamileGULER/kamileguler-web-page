Anı Bırakma Sitesi

Anı bırakma sitesi, QR kod ile erişilebilir ve anılarınızı paylaşabilirsiniz.

×

Bu proje, modern web teknolojileri kullanılarak geliştirilmiş bir anı bırakma sitesidir. Responsive tasarım ile tüm cihazlarda mükemmel görünüm sağlar. HTML5, CSS3 ve JavaScript kullanılarak geliştirilmiş olup GitHub Pages üzerinde yayınlanmıştır. Kullanıcılar anılarını ve fotoğraflarını paylaşabilir.

Proje Görselleri

Özellikler

  • Responsive tasarım (mobil uyumlu)
  • Modern ve temiz arayüz
  • CSS Grid ve Flexbox kullanımı
  • Smooth scroll ve animasyonlar
  • SEO optimizasyonu
  • Hızlı yükleme süreleri

RAG Chatbot

Python ile geliştirilen akıllı chatbot

×

RAG (Retrieval-Augmented Generation) teknolojisi kullanılarak geliştirilmiş akıllı chatbot projesi. Bu chatbot, belirli bir veri kaynağından bilgi alarak kullanıcı sorularına doğru ve güncel yanıtlar verebilmektedir. Python, Pandas ve modern NLP kütüphaneleri kullanılarak geliştirilmiştir.

Proje Görselleri

Özellikler

  • RAG teknolojisi entegrasyonu
  • Doğal dil işleme (NLP)
  • Veri analizi ve işleme
  • Web tabanlı arayüz
  • Admin paneli
  • Çok dilli destek

Todo List Uygulaması

React ve FastAPI ile geliştirilen modern todo uygulaması

×

Modern web teknolojileri kullanılarak geliştirilmiş tam özellikli todo list uygulaması. Frontend'de React ve Vite, backend'de Python FastAPI kullanılmıştır. Kullanıcılar görevlerini ekleyebilir, düzenleyebilir ve takip edebilir. Responsive tasarım ile tüm cihazlarda kullanılabilir.

Proje Görselleri

Özellikler

  • Modern React uygulaması
  • FastAPI backend
  • Responsive tasarım
  • Karanlık/Açık tema desteği
  • Gerçek zamanlı güncelleme
  • CRUD işlemleri
